# Islamic Prayer Times Wallpaper Generator

**Bismillah** - Generate beautiful lockscreen wallpapers with accurate weekly prayer times.

No more photographing calendars. Just open, pick your city, choose a theme, and download.

## Features

- **6 Prayer Times** - Fajr, Sunrise, Dhuhr, Asr, Maghrib, Isha
- **10 Themes** - Islamic dark/light, modern gradients, and a shuffle button for endless variants
- **Weekly / Bi-weekly Views** - Toggle between 7 or 14 day layouts
- **12hr / 24hr Format** - Clean AM/PM group separators in 12hr mode
- **Offline First** - Full year cached in IndexedDB, works without internet
- **Any City** - Configurable location with popular city presets
- **PWA Ready** - Install on your phone, use like a native app
- **1080x1920 PNG** - Pixel-perfect for phone lockscreens

## Quick Start

```bash
npm install
npm run dev
```

## Tech

React + TypeScript + Vite + Aladhan API + IndexedDB + PWA
